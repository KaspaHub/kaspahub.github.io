Kaspa Donations Privacy Policy

Kaspa Donations is a browser extension designed to help users discover and use Kaspa donation addresses on websites. This Privacy Policy explains what data we collect, how we use it, and your rights.

1. Data We Collect

Kaspa Donations is built from the ground up with privacy in mind. The extension does not collect, store, or transmit any personal or sensitive information. We follow a strict privacy philosophy that includes:

  • No advertising - The extension contains no ads or ad-related code.
  • No telemetry - We do not monitor, record, or report any usage data.
  • No tracking - It doesn't track you or your activity in any way.
  • No AI intrusion - There is no AI involvement in analyzing your data, behavior, or interactions.

Specifically:

  • We do not collect your IP address, browsing history, or any personally identifiable information.
  • We do not send any data to external servers or third parties.
  • We do not use cookies, analytics, trackers, or fingerprinting techniques.

2. How the Extension Works

Kaspa Donations operates entirely within your browser. When active, it does the following:

  • Injects a script on web pages you visit (limited to HTTP/HTTPS pages) to look for specific <meta> tags: kaspa:wallet and optionally kaspa:title.
  • If a valid Kaspa address is found, it displays the address and generates a QR code for your convenience. All of this happens locally in your browser session.
  • It uses icon changes and popup behavior to give visual feedback but does not track user actions or interactions.

3. Permissions

To function properly, the extension requests the following minimal permissions:

  • activeTab - Allows the extension to run only on the currently active browser tab when you activate it.
  • scripting - Enables the extension to inject and execute a script that reads the meta tags from the page.

4. Content Script Behavior

The extension includes a content script that runs on all HTTP and HTTPS pages. This script checks for Kaspa-related meta tags and sends a message to the background script to update the extension icon accordingly. No data is stored or transmitted beyond this internal messaging.

5. Changes to This Policy

If we make changes to this Privacy Policy, they will be posted here with updated date.

6. Contact

If you have any questions or concerns about this privacy policy, please reach out at https://x.com/KaspaHub.
